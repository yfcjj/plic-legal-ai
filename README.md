# 诉前智检 (PLIC) - 智能诉讼准备助手

## 📋 项目简介

**诉前智检 (Pre-Litigation Intelligent Checker)** 是一款面向法律实务的智能辅助工具，专注于**金融借款合同纠纷**和**物业服务合同纠纷**的诉前准备工作。

系统能够从案情描述中自动抽取关键要素，识别被告，确定管辖法院，生成符合规范的要素式起诉状，大幅提升律师和法律工作者的办案效率。

## 🎯 核心功能

### 1. 双案由支持
- ✅ **金融借款合同纠纷**：借款本金、利率、逾期状态识别
- ✅ **物业服务合同纠纷**：物业费、欠费期间、服务状态识别

### 2. 智能要素抽取
- 🎯 **被告识别**：自动识别并提取被告姓名/名称
- 🏛️ **管辖判断**：根据案情自动建议管辖法院
- 📊 **要素结构化**：将案情文本转换为结构化JSON数据

### 3. 文书自动生成
- 📄 **要素式起诉状**：按照最高人民法院示范文本格式生成
- ⚖️ **诉讼请求**：根据案情自动生成标准诉请
- 📋 **证据清单**：提示应提交的证据材料

## 🚀 快速启动

### 方法一：一键启动（推荐）

1. 双击 `start.bat` 文件
2. 等待后端服务启动（看到 `Uvicorn running on http://0.0.0.0:8000`）
3. 用浏览器打开 `index.html` 文件

### 方法二：手动启动

#### 1. 安装Python依赖
```bash
pip install fastapi uvicorn
```

#### 2. 启动后端服务
```bash
python main.py
```
或
```bash
uvicorn main:app --reload --port 8000
```

#### 3. 打开前端页面
直接用浏览器打开 `index.html` 文件

## 🔧 技术架构

### 后端技术栈
- **FastAPI**：现代化、高性能的Python Web框架
- **Uvicorn**：ASGI服务器
- **正则表达式**：要素抽取规则引擎
- **Pydantic**：数据验证

### 前端技术栈
- **Vue 3**：渐进式JavaScript框架
- **Element Plus**：Vue 3 组件库
- **原生Fetch API**：HTTP请求

### 设计特点
- 🎨 **前后端分离**：前端通过RESTful API与后端通信
- 🔒 **CORS配置**：支持跨域访问
- 📱 **响应式设计**：适配不同屏幕尺寸
- ⚡ **实时处理**：毫秒级响应速度

## 📊 API接口文档

### 1. 生成要素式起诉状
**POST** `/api/generate`

请求体：
```json
{
  "case_type": "finance",  // 或 "property"
  "text": "案情描述文本"
}
```

响应：
```json
{
  "defendant": "被告名称",
  "court": "管辖法院",
  "elements": {
    // 结构化要素
  },
  "complaint": "要素式起诉状文本",
  "claims": ["诉讼请求1", "诉讼请求2", "..."]
}
```

### 2. 获取测试用例
**GET** `/api/test-cases`

返回内置的测试用例列表
