## 📖 核心概念

### 优先级体系
- **P0（必填）**：起诉法定必备条件，缺少可能导致不予受理
- **P1（建议）**：辅助案件审理的重要信息
- **P2（可选）**：辅助性或说明性信息

### 提取策略
1. **规则提取**：快速正则匹配（前端 JavaScript）
2. **AI 增强**：腾讯元器智能体（后端 Python）
3. **混合模式**：规则初筛 + AI 补充

### 字段分类
- **当事人信息**：原告、被告、代理人
- **诉讼请求**：本金、利息、违约金、诉讼费
- **事实与理由**：合同、期限、标准、催缴、证据
- **具状信息**：签字、日期

## 🔧 技术栈

### 前端
- Vue 3 Composition API
- Element Plus UI 组件
- 原生 JavaScript 正则表达式

### 后端
- FastAPI (Python)
- 腾讯元器智能体 API
- Pydantic 数据验证

### 数据流转
```
用户输入 → 前端规则提取 → 后端 AI 增强 → 表单填充 → 打印导出
```
