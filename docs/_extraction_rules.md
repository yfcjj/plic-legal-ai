# PLIC 要素提取规则文档

## 项目概述
**项目名称**：诉前智检系统 (PLIC - Pre-Litigation Intelligent Checker)
**功能**：从传统起诉状文本中智能提取要素，填充要素式起诉状表格
**支持案由**：金融借款合同纠纷、物业服务合同纠纷

---

## 一、提取规则体系

### 1.1 优先级说明

| 优先级 | 标识 | 说明 | 处理策略 |
|--------|------|------|----------|
| P0 | 必填 | 起诉法定必备条件，缺少可能导致不予受理 | 提取失败需提示用户补充 |
| P1 | 建议 | 辅助案件审理的重要信息 | 尝试提取，失败可容忍 |
| P2 | 可选 | 辅助性或说明性信息 | 可不提取 |

## 二、金融借款合同纠��提取规则

### 2.1 具状信息 (P0)

#### 具状人签字/盖章
- **正则表达式**：`/具状人[（(][^）)]*[）)][：:]\s*([^\n，,。；]+)/`
- **字段映射**：`fin.plaintiffSign` 或 `factClaim.plaintiff`
- **提示**：优先提取法人盖章信息

#### 起诉日期
- **正则表达式**：`/日期[：:]\s*(\d{4})\s*年\s*(\d{1,2})\s*月\s*(\d{1,2})\s*日/`
- **字段映射**：`fin.signDate`
- **格式要求**：组合为 `YYYY年MM月DD日`

---

### 2.2 原告信息 (P0)

#### 原告-自然人
- **姓名**：`/原告[：:]\s*([\u4e00-\u9fa5]{2,4})/` → `fin.fLenderName`
- **住所地**：`/住所地[：:]\s*([^\n，,。；]+)/` → `fin.fLenderAddr`
- **证件号码**：`/(1[3-9]\d{9}|\d{15,18})/` → `fin.fLenderIdNo`

#### 原告-法人/非法人组织
- **名称**：`/原告[：:]\s*([^\n，,。；（【]+)/` → `fin.fLenderName`
- **住所地**：`/住所地[：:]\s*([^\n，,。；]+)/` → `fin.fLenderAddr`
- **注册地**：`/注册地[：:]\s*([^\n，,。；]+)/` → `fin.fLenderRegAddr`
- **法定代表人**：`/法定代表人[：:]\s*([\u4e00-\u9fa5]{2,4})/` → `fin.fLegalRep`
- **职务**：`/职务[：:]\s*([\u4e00-\u9fa5]{2,4})/` → `fin.fLegalRepPosition`
- **联系电话**：`/电话[：:]\s*([0-9-]+)/` → `fin.fLenderPhone`
- **统一社会信用代码**：`/统一社会信用代码[：:]\s*([A-Z0-9]+)/i` → `fin.fLenderUscc`
- **主体类型**：
  - `/有限责任公司/` → `fin.fLenderTypeLLC = true`
  - `/股份有限公司/` → `fin.fLenderTypeJointStock = true`
  - `/金融机构/` → `fin.fLenderTypeFinancial = true`

---

### 2.3 被告信息 (P0)

#### 被告-自然人
- **姓名**：`/被告[：:]\s*([\u4e00-\u9fa5]{2,4})/` → `fin.fBorrowerName`
- **性别**：
  - `/[,，]男/` → `fin.dMale = true`
  - `/[,，]女/` → `fin.dFemale = true`
- **出生日期**：`/(\d{4})年(\d{1,2})月(\d{1,2})日/` → `fin.dBirth`
- **民族**：`/(汉族|[\u4e00-\u9fa5]{2})/` → `fin.dNation`
- **住址**：`/住[所址]([^\n，,。；]+)/` → `fin.dAddr`
- **证件号码**：`/(1[3-9]\d{9}[A-Za-z0-9]|\d{15,18}[A-Za-z0-9]?)\b/` → `fin.dIdNo`
- **联系电话**：`/(?:联系电话)?[：:]?\s*(1[3-9]\d{9})/` → `fin.dPhone`

#### 被告-法人/非法人组织
- **名称**：`/被告[：:]\s*([^\n，,。；（【]+)/` → `fin.fBorrowerName`
- **住所地**：`/住所地[：:]\s*([^\n，,。；]+)/` → `fin.fBorrowerAddr`
- **法定代表人**：`/法定代表人[：:]\s*([\u4e00-\u9fa5]{2,4})/` → `fin.fLegalRep`
- **统一社会信用代码**：`/统一社会信用代码[：:]\s*([A-Z0-9]+)/i` → `fin.fBorrowerUscc`

### 2.4 诉讼请求 (P0)

#### 尚欠借款本金
- **正则表达式**：`/(?:本金|借款)[^\n]*?([0-9,]+(?:\.\d+)?)\s*万?元/`
- **字段映射**：`fin.fPrincipalAmount`
- **处理**：去除逗号，如有"万"字则乘以10000

#### 欠付利息、复利、罚息
- **期内利息**：`/期内?[利利息][^\n]*([0-9,]+(?:\.\d+)?)\s*元/` → `fin.fInterestAmount`
- **罚息/违约金**：`/罚息[^\n]*([0-9,]+(?:\.\d+)?)\s*元/` → `fin.fPenaltyAmount`
- **复利**：`/复利[^\n]*([0-9,]+(?:\.\d+)?)\s*元/` → `fin.fCompoundAmount`

#### 是否请求利息支付至实际清偿之日
- **正则表达式**：`/至(?:实际)?清偿(?:之日|日止)/`
- **字段映射**：`fin.payableToEnd = true`

#### 是否要求提前还款/解除合同
- **提前还款**：`/提前还款|加速到期/` → `fin.earlyRepayment = true`
- **解除合同**：`/解除合同/` → `fin.contractTermination = true`

#### 是否主张担保权利
- **正则表达式**：`/担保|抵押|质押/`
- **字段映射**：`fin.claimCollateral = true`

#### 是否主张诉讼费用由被告承���
- **正则表达式**：`/诉讼[费目].*?由被告承担|被告承担.*?诉讼费/`
- **字段映射**：`fin.claimFeeYes = true`

#### 案件标的总额
- **正则表达式**：`/标的总额[：:]\s*([0-9.,]+)\s*元/`
- **字段映射**：`fin.claimTotal`

---

### 2.5 事实与理由 (P0)

#### 借款合同签订情况
- **合同名称**：`/合同[名称称]?[：:]?\s*《([^》]+)》/` → `fin.factContractName`
- **合同编号**：`/编号[：:]?\s*([A-Za-z0-9-]+)/` → `fin.factContractNo`
- **签订时间**：`/签(?:订|署)[时间地]?[：:]?\s*(\d{4}年\d{1,2}月\d{1,2}日)/` → `fin.factContractDate`
- **签订地点**：`/签(?:订|署)[地地?点]?[：:]?\s*([^\n，,。；]+)/` → `fin.factContractLocation`

#### 借款金额
- **约定金额**：`/约定[：:]?\s*([0-9,]+(?:\.\d+)?)\s*万?元/` → `fin.fPrincipalAgreed`
- **实际发放**：`/实际发放[：:]?\s*([0-9,]+(?:\.\d+)?)\s*万?元/` → `fin.fPrincipalActual`

#### 借款期限
- **是否到期**：`/已到期|期限届满/` → `fin.maturedYes = true`
- **借款期间**：`/(\d{4})[年.](\d+)[年.](\d+)[日]?起[至—-](\d{4})[年.](\d+)[年.](\d+)/` 
  - → `fin.loanStartYear/Month/Day`
  - → `fin.loanEndYear/Month/Day`

#### 借款利率
- **期内利率**：`/年利率[：:]?\s*([0-9.]+)%/` → `fin.fInterestRate`
- **逾期利率**：`/逾期[：:]?\s*[^0-9]*([0-9.]+)%/` → `fin.fOverdueRate`
- **复利**：`/复利/` → `fin.compound = true`
- **罚息**：`/罚息|违约金[：:]?\s*[^0-9]*([0-9.]+)%/` → `fin.fPenaltyRate`

#### 借款提供时间
- **正则表达式**：`/(?:放款|借款提供|提供借款)[时间地]?[：:]?\s*(\d{4}年\d{1,2}月\d{1,2}日)/`
- **字段映射**：`fin.fDisbursementDate`

#### 还款方式
- **等额本息**：`/等额本息/` → `fin.fRepayEqualPrincipalInterest = true`
- **等额本金**：`/等额本金/` → `fin.fRepayEqualPrincipal = true`
- **到期一次性**：`/到期.*?一次.*?还本/` → `fin.fRepayOneTime = true`
- **按月计息**：`/按月.*?计息/` → `fin.fRepayMonthlyInterest = true`

#### 还款情况
- **已还本金**：`/已还[：:]?\s*本金[^0-9]*([0-9,]+(?:\.\d+)?)\s*元/` → `fin.fRepaidPrincipal`
- **已还利息**：`/已还[：:]?\s*利息[^0-9]*([0-9,]+(?:\.\d+)?)\s*元/` → `fin.fRepaidInterest`
- **还息截止日**：`/还息至[：:]?\s*(\d{4}年\d{1,2}月\d{1,2}日)/` → `fin.fInterestPaidToDate`

#### 是否存在逾期还款
- **正则表达式**：`/逾期|未还|违约/`
- **字段映射**：`fin.overdueYes = true`

#### 担保信息
- **抵押合同**：`/抵押合同/` → `fin.collateralContract = true`
- **质押合同**：`/质押合同/` → `fin.collateralContract = true`
- **保证合同**：`/保证合同/` → `fin.guaranteeContract = true`
- **保证方式**：
  - `/一般保证/` → `fin.guaranteeGeneral = true`
  - `/连带责任/` → `fin.guaranteeJoint = true`
- **最高额担保**：`/最高额/` → `fin.maxCollateral = true`

### 4.4 质量控制

- **必填字段缺失**：返回警告，提示用户手动补充
- **格式校验**：日期格式、金额格式、电话号码格式
- **逻辑校验**：
  - 原告、被告不能为空
  - 标的总额 = 各项金额之和
  - 日期逻辑：开始日期 < 结束日期
  - 被告出生日期 < 当前日期
