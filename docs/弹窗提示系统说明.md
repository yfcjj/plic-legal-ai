#### 弹窗4: 诉讼时效风险提示 ✅

**触发条件:**
用户填写的欠费截止日期距离当前时间超过3年

**检测逻辑:**
```javascript
if (isOverThreeYears(prop.feeEndDate)) {
    risks.push({
        title: '诉讼时效风险',
        content: '物业费适用诉讼时效（通常3年）；
        超过时效部分法院不予保护；
        必须提供催缴记录、短信、挂号信等证据中断时效。...'
    });
}
```

**辅助函数:**
```javascript
const isOverThreeYears = (dateStr) => {
    const d = parseCnDate(dateStr);
    if (!d) return false;
    return new Date().getTime() - d.getTime() >
           3 * 365 * 24 * 60 * 60 * 1000;
};
```

**弹窗内容:**
```
法律风险提示

3.【诉讼时效风险】
物业费适用诉讼时效（通常3年）；
超过时效部分法院不予保护；
必须提供催缴记录、短信、挂号信等证据中断时效。

AI无法判断时效是否届满，请自行核实。
```

## 🎨 弹窗样式说明

### Element Plus MessageBox配置

```javascript
ElementPlus.ElMessageBox.alert/detect/alert({
    message: '弹窗内容',
    title: '弹窗标题',
    confirmButtonText: '按钮文字',
    cancelButtonText: '取消按钮文字',
    type: 'warning', // 类型: success/warning/info/error
    dangerouslyUseHTMLString: true // 允许HTML内容
});
```

### 样式特点

- **警告类型:** type: 'warning' - 黄色警告图标
- **HTML支持:** dangerouslyUseHTMLString: true - 支持富文本格式
- **双按钮:** confirm/cancel - 用户可选择继���或返回
