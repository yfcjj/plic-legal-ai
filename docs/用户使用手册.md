## 快速上手

### 第一步:打开系统

用浏览器(推荐Chrome或Edge)打开 `index.html` 文件。

### 第二步:选择案由

在"📄 起诉状表格"标签页中,选择案件类型:
- 🏘️ 物业服务合同纠纷
- 🏦 金融借款合同纠纷

### 第三步:输入案情

将传统起诉状或案情描述粘贴到文本框中。

### 第四步:提取要素

点击"🔍 规则提取"按钮,系统自动填充表单。

### 第五步:生成文书

点击"📝 生成要素式起诉状",确认风险提示后打印。

## 物业服务合同纠纷使用指南

### 场景一:从传统起诉状提取

**步骤:**

1. **准备文本**
   ```
   民事起诉状
   
   原告:襄阳津诚物业服务有限公司
   住所地:襄阳市襄州区张湾街道江营北路28号门面房
   法定代表人:李建明,职务:执行董事兼总经理
   联系电话:13907278899
   统一社会信用代码:91420607MA4F3XKW7L
   
   被告:张卫国,男,1973年4月28日出生,汉族
   住址:襄阳市襄州区东方新城小区12栋3单元601室
   联系电话:13871652211
   身份证号:420621197304281537
   
   诉讼请求:
   1. 请求判令被告支付截至2025年12月31日的物业费8078元;
   2. 请求判令被告支付违约金1615.6元;
   3. 请求判令被告承担诉讼费用。
   
   事实与理由:
   被告系东方新城小区12栋3单元601室业主,房屋面积128.23平方米。
   根据《物业管理服务合同》,物业费标准为1.4元/月/㎡。
   被告自2023年4月1日至2025年12月31日未交纳物业费,共45个月。
   具体计算:1.4元/月/㎡×128.23㎡×45个月=8078元。
   违约金按欠费金额的20%计算:8078元×20%=1615.6元。
   原告已通过上门、张贴、短信等方式多次催告,最后一次催收时间为2025年12月20日。
   
   此致
   襄阳市襄州区人民法院
   
   具状人:襄阳津诚物业服务有限公司(盖章)
   日期:2026年3月3日
   ```

2. **粘贴到系统**
   - 选择"🏘️ 物业服务合同纠纷"
   - 粘贴到"案情描述"文本框

3. **点击提取**
   - 点击"🔍 规则提取"
   - 系统自动填充100+字段

4. **核对结果**
   - 检查"当事人信息"是否正确
   - 检查"诉讼请求"金额是否准确
   - 检查"事实与理由"是否完整

5. **生成文书**
   - 点击"📝 生成要素式起诉状"
   - 查看风险提示(如有)
   - 确认后打印

## 常见问题FAQ

### Q1: 提取不准确怎么办?

**A:** 分情况处理:

1. **少量字段错误**
   - 手动修改对应字段
   - 继续生成文书

2. **大量字段错误**
   - 检查文本格式是否符合规范
   - 尝试"AI智能提取"
   - 联系技术支持

3. **特定字段总是提取错误**
   - 截图反馈给我们
   - 我们会优化规则

### Q2: 系统提示"必填字段未填写"?

**A:** 检查以下必填字段:

**物业案件必填:**
- 原告名称
- 被告姓名
- 物业费金额
- 物业坐落位置
- 物业面积
- 物业费标准
- 法律依据

**金融案件必填:**
- 原告名称
- 被告姓名
- 本金金额

### Q3: 打印时横线太短或没有横线?

**A:** 已在最新版本修复,确保使用最新版:
- 文件名包含`20260418`及以后的版本

### Q4: AI智能提取提示"后端服务未启动"?

**A:** 按以下步骤启动后端:

```bash
# 1. 安装依赖
pip install fastapi uvicorn openai python-multipart

# 2. 配置API Key
export OPENAI_API_KEY="your-api-key"

# 3. 启动服务
python main.py
```

### Q5: 如何确认生成的文书正确?

**A:** 核对清单:

- [ ] 当事人信息准确
- [ ] 金额计算无误
- [ ] 日期正确
- [ ] 合同条款引用准确
- [ ] 证据清单完整
- [ ] 签名区信息完整

### Q6: 可以修改生成后的文书吗?

**A:** 可以:

1. **打印前修改**
   - 在表单中直接修改
   - 重新点击"生成"

2. **打印后修改**
   - 导出PDF
   - 使用PDF编辑器修改
   - 或打印后手写修改

### Q7: 如何保存案件信息?

**A:** 当前版本不支持自动保存,建议:
- 截图保存
- 复制文本到Word
- 等待后续版本更新

### Q8: 系统支持哪些浏览器?

**A:** 推荐使用:
- Chrome 90+
- Edge 90+
- Firefox 88+
- Safari 14+

不支持IE浏览器。
